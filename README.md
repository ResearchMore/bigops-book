# 功能特点

### 开源软件统一认证SSO

---

支持广泛的开源软件统一认证，包括Zabbix、Jira、Jenkins、GitLab等，还在不断增加中。

### 千人千面控制台

---

每个用户都拥有自己独立的控制台，可以按照自己的喜好定义，包括桌面背景图片、图标样式、图标位置、图标是否显示等。

还可以定义自己的快捷按键，随心所欲的想去哪里就去哪里。
